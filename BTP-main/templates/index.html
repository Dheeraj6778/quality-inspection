  • Canny’s algorithm is an edge detection algorithm that is used to detect a wide range of edges
  • The first step in this algorithm is noise reduction. This step basically removes noise in an image
  • The next step is to find out edge intensity and direction
  • After this step, we almost get our desired image, but some edges are thick
  • The next step is to thin out the edges.
  • After thinning, we now identify three kinds of pixels: strong, weak, and non-relevant.
  • Strong pixels are pixels having high intensity and they contribute to the final edge
  • Weak pixels are the pixels that have an intensity value that is not enough to be considered strong but not small enough to be considered non-relevant
  • Non-relevant pixels don’t contribute to the final edge
  • So based on the above results, we transform the weak pixel into a strong one if at least one pixel around the pixel being processed is a strong one.
  • Finally we get our desired image

Calculation of model performance parameters from Confusion Matrix

TP is True Positive

TN is True Negative

FP is False Positive

FN is False Negative

Model performance parameters

Accuracy = (TN+TP)/(TN+TP+FN+FP)

Precision = TP/(FP+TP)

Sensitivity or Recall = TP/(TP+FN)

Specificity = TN/(TN+FP)

Calculation of F1 score

What is F1 score and why do we need it?
  • By definition, F1-score is the harmonic mean of precision and recall. It combines precision and recall into a single number using the following formula
  • F1-score = (2*precision*recall)/(precision+recall)
  • F1 scores can range from 0 to 1,
  • Higher F1 scores are generally better
  • F1-scores with values close to 1 are good
